# coffee-json-dsl-cfn-runlevel
coffee-json-dsl plugin to define runlevels of resources of AWS CloudFormation stack templates

## Overview

coffee-json-dsl-cfn-runlevel introduces resource runlevel to AWS CloudFormation stack templates.
This is a plugin of [coffee-json-dsl][https://github.com/rld2drkw/coffee-json-dsl] and designed to use with
[coffee-json-dsl-cfn][[https://github.com/rld2drkw/coffee-json-dsl-cfn]] plugin.
